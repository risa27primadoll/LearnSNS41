signup.php

0 ユーザーは「ユーザー名」「メールアドレス」「パスワード」を入力することができる
0 各種フォームには入力チェック（バリデーション）がある
0 ユーザー名が空だった場合エラーメッセージが出る
0 メールアドレスが空だった場合エラーメッセージが出る
0 パスワードが空だった場合エラーメッセージが出る
0 パスワードが4文字以下だった場合エラーメッセージが出る
0 プロフィール画像が未選択だった場合エラーメッセージが出る
0 プロフィール画像の拡張子が「jpg」「png」「gif」以外だった場合エラーメッセージが出る
0 すべての入力チェックを通過した場合
0 セッションに入力された情報を保存する
0 プロフィール画像をprofile_imagesフォルダへアップロード
0 入力情報確認画面（check.php）へ遷移する
0 メールアドレス重複チェック
0 入力欄の入力内容保持

0 ユーザーは入力情報に間違いがないか確認することができる
0 正規のルートを通らずにcheck.phpへのリクエストが行われた場合はsignup.phpへ強制遷移する
0 入力情報に間違いがあった場合は戻るボタンを押すことでsignup.phpへ戻ることができる
0 ユーザー登録ボタンを押すことで入力情報をデータベースへ登録する
0 登録処理完了後、アカウント作成完了ページ(thanks.php)へ遷移する
